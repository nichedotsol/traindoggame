# Sprite Sheet Coverage

This directory contains transparent PNG sprite sheets plus `manifest.json`.

The rule for this asset pass is: every visible state change has named sprite frames in the manifest.

## Characters

- `train_dog.png`: idle, move, jump, land, duck, hurt, cheer, board
- `passenger_*.png`: idle, walk, wave, board, cheer for conductor, fox, rabbit, cat, bear, frog, sheep, and astronaut passengers

## Collectibles

- `collectible_gold_whistle.png`: idle, spin, sparkle, collect
- `collectible_silver_bone.png`: idle, spin, sparkle, collect
- `collectible_rainbow_ticket.png`: idle, spin, sparkle, collect
- `collectible_star_coin.png`: idle, spin, sparkle, collect

## Obstacles And Enemies

- `hazard_*.png`: idle, warning, active, broken for spiked ball, boulder, cactus, crow, pine rock, snow boulder, lantern, wave crate, ice spike, meteor, and crater

## Destination Portals

- `portal_*.png`: closed, idle, open, activate for forest, mountains, city, desert, ocean, arctic, space, and moon portals

## Environment Motion

- `environment_*.png`: cloud drift, parallax far, parallax mid, parallax near, rail scroll, full scroll, arrival flash, danger shake for every destination

## UI And Feedback

- `ui_hearts.png`: full, empty, pulse, gain, lose
- `ui_meters.png`: whistle, bone, coin, space button, route marker
- `fx_smoke.png`: puff, burst, fade
- `terrain_tiles.png`: animated/variant tiles and props for grass, stone, bush, rail, wood, water, sand, ice, flower, mushroom, rock, crate, barrel, sign, lamp, and cloud
- `destination_*.png`: idle, selected, complete for every route destination icon
