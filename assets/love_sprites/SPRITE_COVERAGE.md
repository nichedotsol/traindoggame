# LÖVE2D Sprite Coverage

This is the primary TrainDog sprite atlas for the LÖVE2D revamp.

Generated by:

```powershell
python tools/build_love_sprites.py
```

The atlas contains **257 individual PNG sprite sheets** plus `manifest.lua`.

Each sheet represents one subject and one action/state, with four animation frames arranged horizontally. The game loads these sheets through `src/atlas.lua`.

## Coverage

- TrainDog: 8 movement/action sheets
- Passengers: 8 characters x 5 actions = 40 sheets
- Collectibles/artifacts: 4 items x 4 states = 16 sheets
- Hazards/enemies: 11 hazards x 4 states = 44 sheets
- Destination portals: 8 portals x 4 states = 32 sheets
- Destination environments: 8 environments x 8 motion/state sheets = 64 sheets
- Route destination icons: 8 destinations x 3 states = 24 sheets
- Terrain/tools/props: 16 variant sheets
- FX: smoke puff, burst, fade = 3 sheets
- HUD: heart and meter animation states = 10 sheets

Total: 257 sheets.
